Brain Challenges

MapleStory key binding system

Unity C#

A keybinding system that replicates the keybinding system in MapleStory. You have a pool of function keys, i.e., STATS and KEYBINDING, and you can set them to shortcuts you want. You can save or cancel the changes or reset all keys to the pool. Additionally, the system is inter-connect with slots in the bottom-right corner that you can change either place and it will apply for both.
